Welcome to the GitHub page for Coitus Cupid.

Coitus Cupid is a proof-of-concept dating website which uses the latest and greatest web technologies. Written in Rails 3.2 and Ruby 1.9.3, we aim to provide a reference quality web application which is fully regression tested, secure and fast. Please browse the issues tracker for current progress and milestones.

Usage

The code is deployed and available at http://coitusproject.herokuapp.com/. Please note that as of April 2012, the site is currently still in development and as such things may break or become unusable.
